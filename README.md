# Tic Tac Toe

## Prerequisites

* Python 3.7

## Setup

Install dependencies:

```
$ pipenv install
```

If `pipenv` is not installed on your system, you can easily install it with `pip install pipenv`.

## Running the Tests

```
$ python3 run_tests.py
```

## Playing the Game

```
$ python3 startgame.py
```
